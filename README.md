## Get Started

### Installation

`$ yarn add react-native-common`
or
`$ npm install react-native-common --save`

## Usage

```javascript
import { Button } from 'react-native-common';

<Button
  style={{width: 270}}
  borderRadius={10}
  backgroundColor="red"
  label={`Welcome to\nReact Native Common`}
/>
```
